01. 함수 : 선언적 함수

선언적 함수(declarative function)는 명령형 프로그래밍의 반대 개념으로, 어떻게 해야 하는지를 명시하는 대신에 원하는 결과를 설명하는 방식으로 작성된 함수를 의미합니다. 즉, 선언적 함수는 함수의 구현에 대한 세부 정보를 명시적으로 제공하지 않고도 작업을 수행할 수 있도록 해줍니다. 

{

function func() {
    console.log("01. 함수가 실행되었습니다.")
}
func();

}
