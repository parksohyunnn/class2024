07. indexOf()

let fruits = ["사과", "바나나", "딸기", "포도", "바나나"];

  • JavaScript의 배열 객체의 indexOf() 메서드는 배열 내에서 지정된 요소의 첫 번째 인덱스를 반환합니다.
  • 이 메서드는 배열을 순회하면서 해당 요소와 일치하는 첫 번째 요소의 인덱스를 찾아 반환합니다.
  • 만약 배열 내에 해당 요소가 없으면 -1을 반환합니다. 

{
console.log(fruits.indexOf("바나나")); // 출력: 1
}

여기서 "바나나"는 배열 fruits에서 두 번째 요소이므로 인덱스 1을 반환합니다.
또한, indexOf() 메서드는 두 번째 매개변수로 시작 인덱스를 지정할 수 있습니다.
이를 통해 배열의 특정 위치부터 검색을 시작할 수 있습니다.
만약 시작 인덱스를 지정하지 않으면 기본값은 0으로 설정됩니다.
이 매개변수를 사용하여 다음과 같이 특정 위치 이후부터 요소를 찾을 수 있습니다: 

{
console.log(fruits.indexOf("바나나", 2)); // 출력: 4
}

이 예제에서는 배열 fruits에서 두 번째 "바나나" 이후의 요소를 찾으므로, 네 번째 요소의 인덱스인 4를 반환합니다.
만약 찾으려는 요소가 배열에 여러 번 나타나더라도 indexOf()는 항상 첫 번째로 일치하는 요소의 인덱스만 반환합니다.

10. match()

string.match(regexp)

  • string: 일치 여부를 확인할 대상 문자열입니다.
  • regexp: 검색할 정규 표현식입니다.
  • 이 메서드는 주어진 정규 표현식 regexp과 문자열 string을 비교하여 일치하는 부분을 찾습니다.
  • 이때, regexp가 일치하는 부분을 찾을 수 없으면 null을 반환합니다.
  • 일치하는 부분이 있다면, match()는 일치하는 부분을 담은 배열을 반환합니다.

{
const str = "Hello World!";
const result = str.match(/Hello/);
console.log(result); // ["Hello"]
}

문자열에 대한 일치 여부 확인: 이 예제에서는 "Hello"라는 부분 문자열이 문자열 str에 있는지 확인하고, 일치하는 부분 문자열을 배열로 반환합니다.

{
const str = "2024-05-09";
const result = str.match(/(\d{4})-(\d{2})-(\d{2})/);
console.log(result); // ["2024-05-09", "2024", "05", "09"]
}

이 예제에서는 날짜 형식의 문자열에서 연도, 월, 일에 해당하는 부분을 추출하기 위해 그룹화된 정규 표현식을 사용합니다.
match() 메서드는 정규 표현식과 일치하는 전체 문자열과 그룹에 해당하는 부분 문자열을 배열로 반환합니다.
match() 메서드는 일치하는 부분이 없을 경우 null을 반환합니다. 이 때문에 반환값을 검사하여 일치하는 부분이 있는지 확인하는 것이 좋습니다.
또한, match() 메서드는 정규 표현식의 g 플래그를 사용하여 전역 검색을 수행할 수 있습니다. 이 경우에는 일치하는 모든 부분을 배열로 반환합니다.

16. replace()

string.replace(regexp|substr, newSubstr|function)

  • string: 대상 문자열입니다.
  • regexp|substr: 교체할 부분을 지정하는 정규 표현식이나 문자열입니다.
  • newSubstr|function: 대체할 새로운 문자열이나 대체를 수행하는 함수입니다.
  • 이 메서드는 string에서 regexp 또는 substr과 일치하는 첫 번째 부분을 찾아 newSubstr 또는 함수에 의해 대체합니다.
  • 그리고 새로운 문자열로 바뀐 문자열을 반환합니다.
  • 만약 regexp로 정규 표현식을 사용했다면, 일치하는 부분이 여러 개라도 첫 번째 일치하는 부분만 대체됩니다.
  • 이를 전체 대체하려면 정규 표현식에 g 플래그를 추가해야 합니다.

{
const str = "Hello World!";
const newStr = str.replace("World", "Universe");
console.log(newStr); // "Hello Universe!"
}

위 예제에서 replace() 메서드는 문자열 "Hello World!"에서 "World"라는 부분 문자열을 찾아 "Universe"로 대체하여 새로운 문자열 "Hello Universe!"를 반환합니다.

19. slice()

array.slice(begin, end)

  • array: 원본 배열입니다.
  • begin: 추출을 시작할 인덱스입니다.
  • 이 인덱스는 결과에 포함됩니다.
  • 음수를 사용할 경우, 배열의 끝에서부터의 상대적 위치를 나타냅니다.
  • 만약 생략하면 0으로 간주됩니다.
  • end: 추출을 종료할 인덱스입니다.
  • 이 인덱스는 결과에 포함되지 않습니다.
  • 음수를 사용할 경우, 배열의 끝에서부터의 상대적 위치를 나타냅니다.
  • 만약 생략하면 배열의 길이로 간주됩니다.
  • slice() 메서드는 원본 배열의 일부분을 추출하여 새로운 배열을 반환합니다.
  • 추출할 부분은 시작 인덱스부터 종료 인덱스 직전까지의 요소입니다. 원본 배열은 변경되지 않습니다.

{
const array = [1, 2, 3, 4, 5];
const newArray = array.slice(1, 4);
console.log(newArray); // [2, 3, 4]
}

위 예제에서 slice() 메서드는 배열 array의 인덱스 1부터 4 직전까지의 요소를 추출하여 새로운 배열 newArray를 반환합니다.
따라서 결과는 [2, 3, 4]입니다. 원본 배열 array는 변경되지 않습니다.

20. substring()

string.substring(startIndex, endIndex)

  • string: 부분 문자열을 추출할 원본 문자열입니다.
  • startIndex: 추출을 시작할 인덱스입니다.
  • 이 인덱스는 결과에 포함됩니다.
  • 음수를 사용하면 0으로 처리됩니다.
  • endIndex: 추출을 종료할 인덱스입니다.
  • 이 인덱스는 결과에 포함되지 않습니다. 생략될 경우, string의 길이가 사용됩니다.
  • 음수를 사용하면 0으로 처리됩니다.
  • substring() 메서드는 string에서 startIndex부터 endIndex 직전까지의 부분 문자열을 추출하여 반환합니다.
  • 이 때, startIndex와 endIndex가 문자열의 길이를 초과하는 경우 자동으로 문자열의 끝에 맞추어집니다.
  • 또한, startIndex가 endIndex보다 클 경우, 두 인덱스가 서로 바뀌어 처리됩니다.

{
const str = "Hello World!";
const subStr = str.substring(6, 11);
console.log(subStr); // "World"
}

위 예제에서 substring() 메서드는 문자열 "Hello World!"에서 인덱스 6부터 11 직전까지의 부분 문자열인 "World"를 추출하여 반환합니다. 결과는 "World"입니다.

22. split()

string.split(separator, limit)

  • string: 분할할 대상 문자열입니다.
  • separator: 문자열을 나눌 구분자입니다.
  • 이 구분자를 기준으로 문자열이 나누어집니다.
  • 구분자는 문자열 또는 정규 표현식이 될 수ㅡ있습니다. 생략하면 배열에 문자열 전체가 포함됩니다.
  • limit: 반환할 배열의 최대 길이입니다. 생략하면 모든 발생을 포함하는 배열이 반환됩니다.
  • split() 메서드는 string을 separator로 분할하여 각 부분 문자열로 이루어진 배열을 반환합니다.
  • 구분자로 사용된 문자열은 결과에 포함되지 않습니다.

{
const str = "apple,banana,grape,orange";
const arr = str.split(",");
console.log(arr); // ["apple", "banana", "grape", "orange"]
}

위 예제에서 split() 메서드는 문자열 "apple,banana,grape,orange"을 쉼표 ,를 기준으로 분할하여 각 부분 문자열로 이루어진 배열을 반환합니다.
결과는 ["apple", "banana", "grape", "orange"]입니다.

30. trim()

string.trim()

  • string: 공백을 제거할 대상 문자열입니다.
  • trim() 메서드는 문자열의 양 끝에서 공백을 제거한 새로운 문자열을 반환합니다.
  • 이 때, 공백은 공백 문자(space), 탭 문자(tab), 줄 바꿈 문자(newline) 등이 포함됩니다.

{
const str = "   Hello World!   ";
const trimmedStr = str.trim();
console.log(trimmedStr); // "Hello World!"
}

위 예제에서 trim() 메서드는 문자열 " Hello World! "의 양 끝에서 공백을 제거하여 "Hello World!"를 반환합니다.
trim() 메서드는 문자열의 양 끝에서 공백을 제거한 새로운 문자열을 반환합니다. 이 때, 공백은 공백 문자(space), 탭 문자(tab), 줄 바꿈 문자(newline) 등이 포함됩니다.
위 예제에서 trim() 메서드는 문자열 " Hello World! "의 양 끝에서 공백을 제거하여 "Hello World!"를 반환합니다.
