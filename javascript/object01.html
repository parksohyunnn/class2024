01. at()

"문자열".at(위치값)"

  • 지정한 숫자(index)를 받아, 문자열에서 해당되는 인덱스의 요소를 반환합니다.
  • 대소문자를 구분합니다.
  • 문자열의 길이보다 큰 인덱스를 지적하면 undefined를 반환합니다.
  • 음수인 경우 문자열의 오른쪽에서부터 인덱스를 셉니다.

{
    "javascript reference".at()       //j
    "javascript reference".at(0)      //j
    "javascript reference".at(1)      //a
    "javascript reference".at(2)      //v
    "javascript reference".at(3)      //a
    "javascript reference".at(4)      //s
    "javascript reference".at(5)      //c
    "javascript reference".at(-1)     //e
    "javascript reference".at(-2)     //c
    "javascript reference".at(-3)     //n
    "javascript reference".at(-4)     //e
    "javascript reference".at(-5)     //r
    "javascript reference".at(100)    // undefined
}

06..includes()

"문자열".includes(위치값)

  • 특정 문자열이 다른 문자열 내에 포함되어 있는지 여부를 확인합니다.
  • 문자열이 포함되어 있으면 true를, 그렇지 않으면 false를 반환합니다.
  • 문열의 길이보다 큰 인덱스를 지정하면 undefiend를 반환합니다.
  • 대소문자를 구별합니다.

{const str = 'Hello World';

console.log(str.includes('World')); // true
console.log(str.includes('world')); // false

// 위치값을 지정하여 검색 시작 위치를 변경할 수 있습니다.
console.log(str.includes('Hello', 6)); // false
console.log(str.includes('Hello', 0)); // true
}

위 예시에서는 각각 "Hello World" 문자열에 'World'와 'world' 문자열이 포함되어 있는지 여부를 확인하고 있습니다. 또한 위치값을 지정하여 검색 시작 위치를 변경하여 검색 결과를 확인할 수 있습니다.

07. .indexOf()

"문자열".indexOf(찾고자하는 문자열, [위치값])

  • 찾고자 하는 문자열이 발견되면 해당 인덱스를 반환하고, 발견되지 않으면 -1을 반환합니다.
  • 문자열 내에서 특정 문자열의 첫 번째 인덱스를 반환합니다.
  • 특정 문자열이 포함되어 있는지 여부를 확인하는 데 사용됩니다.

console.log(str.includes('Hello', 0)); // true

위 예시에서는 각각 "Hello World" 문자열에 'World'와 'world' 문자열이 포함되어 있는지 여부를 확인하고 있습니다. 또한 위치값을 지정하여 검색 시작 위치를 변경하여 검색 결과를 확인할 수 있습니다.

10. .match()

"문자열".match(검색할 정규식 패턴)

  • 문자열에서 정규식과 일치하는 부분을 찾아 배열로 반환합니다.
  • 정규식과 일치하는 부분을 배열로 반환합니다. 일치하는 부분이 없으면 null을 반환합니다.
  • 정규식에 /g 플래그를 사용하면 모든 일치 항목을 반환합니다.
  • 배열의 첫 번째 요소는 전체 일치 문자열이며, 이후 요소는 캡처 그룹에 해당하는 부분 문자열입니다.

{
const str = 'The rain in Spain falls mainly on the plain';

// 정규 표현식에 매칭되는 부분을 검색하여 배열로 반환합니다.
const result = str.match(/ain/g);

console.log(result); // ["ain", "ain", "ain"]
}

위 예시에서는 문자열 "The rain in Spain falls mainly on the plain"에서 정규 표현식 /ain/g와 매칭되는 부분을 검색하여 배열로 반환하고 있습니다. 결과로는 ["ain", "ain", "ain"] 배열이 반환되었습니다.

16. .replace()

"문자열".replace(교체할 대상을 나타내는 정규 표현식 또는 문자열, 새로운 문자열)

  • 문자열 내에서 searchValue에 해당하는 부분을 찾아 newValue로 교체한 새로운 문자열을 반환합니다.
  • searchValue가 정규 표현식으로 지정된 경우, 문자열 내에서 첫 번째로 일치하는 부분만 교체됩니다. 전체 문자열을 대상으로 검색합니다.
  • 기본적으로 첫 번째로 매치되는 부분만 교체하고, 전역 교체를 원하는 경우 정규 표현식에 g 플래그를 사용해야 합니다.
  • replace() 메서드는 원본 문자열을 변경하지 않고 새로운 문자열을 반환합니다.

{
const str = "The quick brown fox jumps over the lazy dog.";

const replacedStr = str.replace("fox", "cat");
console.log(replacedStr); // "The quick brown cat jumps over the lazy dog."

const replacedWithRegex = str.replace(/brown/, "black");
console.log(replacedWithRegex); // "The quick black fox jumps over the lazy dog."
                                        
}

위 예시에서는 .replace() 메서드를 사용하여 문자열에서 특정 부분을 교체하고 있습니다. 첫 번째 호출에서는 "fox" 문자열을 "cat"으로 교체하고 있으며, 두 번째 호출에서는 /brown/에 매치되는 부분을 "black"으로 교체하고 있습니다.

18. .search()

"문자열".search(검색할 문자열 또는 정규 표현식)

  • search() 메서드는 문자열 내에서 searchValue에 해당하는 부분을 찾아 첫 번째로 일치하는 항목의 인덱스를 반환합니다. 일치하는 항목이 없으면 -1을 반환합니다.
  • searchValue가 문자열인 경우, 전체 문자열을 대상으로 해당 문자열이 처음 나타나는 위치의 인덱스를 반환합니다.
  • searchValue가 정규 표현식인 경우, 정규 표현식에 매칭되는 첫 번째 항목의 인덱스를 반환합니다.
  • 메서드는 전역 검색을 지원하지 않으며, 첫 번째로 매치되는 부분만 반환합니다.

{
                                        const str = "The quick brown fox jumps over the lazy dog.";

                                        console.log(str.search("brown")); // 10
                                        console.log(str.search(/fox/)); // 16
                                        console.log(str.search("cat")); // -1
}

위 예시에서는 .search() 메서드를 사용하여 문자열에서 특정 문자열 또는 정규 표현식에 일치하는 첫 번째 항목의 인덱스를 찾고 있습니다. 첫 번째 호출에서는 "brown" 문자열의 인덱스를 반환하고 있으며, 두 번째 호출에서는 /fox/에 일치하는 부분의 인덱스를 반환하고 있습니다. 세 번째 호출에서는 "cat" 문자열에 대해 일치하는 부분을 찾지 못해 -1을 반환합니다.

19. .slice()

"문자열".slice(추출을 시작할 인덱스를 나타내는 정수 값, 추출을 종료할 인덱스를 나타내는 정수 값)

  • slice() 메서드는 문자열이나 배열에서 지정한 인덱스 범위에 해당하는 부분을 추출하여 새로운 문자열이나 배열을 반환합니다.
  • startIndex부터 endIndex 직전까지의 문자열을 추출합니다. endIndex는 추출되는 문자열에 포함되지 않습니다.
  • startIndex이 endIndex보다 큰 경우, 빈 문자열이 반환됩니다.
  • 음수 인덱스를 사용할 경우, 해당 인덱스는 문자열 끝에서부터의 오프셋을 나타냅니다. 예를 들어, -1은 문자열의 마지막 문자를 나타냅니다.

{
const str = "Hello, world!";

console.log(str.slice(7)); // "world!"
console.log(str.slice(0, 5)); // "Hello"
console.log(str.slice(-6)); // "world!"
console.log(str.slice(7, -1)); // "world"
}

위 예시에서는 .slice() 메서드를 사용하여 문자열에서 지정한 인덱스 범위에 해당하는 부분을 추출하고 있습니다. 첫 번째 호출에서는 인덱스 7부터 끝까지의 부분을 추출하고 있으며, 두 번째 호출에서는 인덱스 0부터 5 직전까지의 부분을 추출하고 있습니다. 세 번째 호출에서는 끝에서부터 6번째 문자부터 끝까지의 부분을 추출하고 있으며, 네 번째 호출에서는 인덱스 7부터 끝에서 두 번째 문자 직전까지의 부분을 추출하고 있습니다.

20. .substring()

"문자열".substring(추출을 시작할 인덱스를 나타내는 정수 값, 추출을 종료할 인덱스를 나타내는 정수 값)

  • 문자열에서 지정한 인덱스 범위에 해당하는 부분을 추출하여 새로운 문자열을 반환합니다.
  • startIndex과 endIndex는 각각 추출의 시작과 끝을 나타냅니다. 이 때, startIndex은 항상 endIndex보다 작거나 같아야 합니다.
  • substring() 메서드는 양수 인덱스만 사용 가능합니다. 음수 값은 0으로 처리됩니다.
  • startIndex과 endIndex를 바꿔도 동일한 결과가 반환됩니다.
  • startIndex과 endIndex는 문자열의 길이를 초과할 경우, 해당 부분을 문자열의 끝으로 처리합니다.

{
const str = "Hello, world!";

console.log(str.substring(7)); // "world!"
console.log(str.substring(0, 5)); // "Hello"
console.log(str.substring(7, 12)); // "world"
}

위 예시에서는 .substring() 메서드를 사용하여 문자열에서 지정한 인덱스 범위에 해당하는 부분을 추출하고 있습니다. 첫 번째 호출에서는 인덱스 7부터 끝까지의 부분을 추출하고 있으며, 두 번째 호출에서는 인덱스 0부터 5 직전까지의 부분을 추출하고 있습니다. 세 번째 호출에서는 인덱스 7부터 12 직전까지의 부분을 추출하고 있습니다.

22. .split()

"문자열".split(나눌 대상이 되는 문자열, 문자열을 나눌 때 사용할 구분자)

  • 문자열을 지정한 구분자를 기준으로 나누어 배열로 반환합니다.
  • separator가 정규 표현식으로 지정될 경우, 해당 정규 표현식에 매칭되는 부분을 기준으로 문자열을 나눕니다.
  • slimit를 지정하면 반환되는 배열의 길이가 최대한으로 제한됩니다. 초과하는 부분은 버려집니다.
  • separator가 생략되거나 빈 문자열 ""로 지정된 경우, 문자열 전체를 한 번에 하나의 요소로 하는 배열을 반환합니다.

{
const str = "apple,banana,grape,orange";

console.log(str.split(",")); // ["apple", "banana", "grape", "orange"]
console.log(str.split(",", 2)); // ["apple", "banana"]
console.log(str.split("")); // ["a", "p", "p", "l", "e", ",", "b", "a", "n", "a", "n", "a", ",", "g", "r", "a", "p", "e", ",", "o", "r", "a", "n", "g", "e"]
console.log(str.split(/\W+/)); // ["apple", "banana", "grape", "orange"]
}

위 예시에서는 .split() 메서드를 사용하여 문자열을 지정한 구분자를 기준으로 나눈 배열을 생성하고 있습니다. 첫 번째 호출에서는 ,를 구분자로 사용하여 문자열을 나누고 있으며, 두 번째 호출에서는 최대 2개의 요소까지만 반환하도록 제한하고 있습니다. 세 번째 호출에서는 빈 문자열을 구분자로 사용하여 문자열의 각 문자를 요소로 하는 배열을 생성하고 있습니다. 마지막 호출에서는 정규 표현식 /W+/을 사용하여 문자열을 나누고 있으며, 구두점이 아닌 문자를 기준으로 배열을 생성하고 있습니다.

30. .trim()

"문자열".trim(공백을 제거할 대상이 되는 문자열)

  • trim() 메서드는 문자열의 양쪽 끝에 있는 공백(스페이스, 탭, 줄 바꿈 등)을 모두 제거한 새로운 문자열을 반환합니다.
  • 원본 문자열은 변경되지 않고, 제거된 공백이 포함된 새로운 문자열이 반환됩니다.

{
const str = "   Hello, world!   ";

console.log(str.trim()); // "Hello, world!"
}

위 예시에서는 .trim() 메서드를 사용하여 문자열의 양쪽 끝에 있는 공백을 제거한 후 결과를 출력하고 있습니다. 출력된 문자열은 "Hello, world!"입니다.
